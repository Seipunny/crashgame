# Product Requirements Document: Прототип Креш Игры для Unity

## Обзор проекта
Создание упрощенного прототипа креш-игры для казино с правильной математикой и RTP 96% в Unity 2022.3 LTS.

## Целевые платформы
- WebGL (основная)
- Android (мобильная версия)
- iOS (мобильная версия)

## Основные требования

### 1. Математическая модель
- **RTP**: 96% (ε = 0.04)
- **Формула краша**: M = (1 - ε) / (1 - U), где U ~ Uniform(0,1)
- **Ограничения**: MIN = 1.01, MAX = 1000
- **Рост мультипликатора**: Экспоненциальная функция с динамическим ускорением
- **Контрольные точки**: x2.00 за 8с, x20.00 за 35с, x100.00 за 50с

### 2. Игровая механика
- Игрок делает ставку перед раундом
- Мультипликатор растет от x1.00
- Игрок может забрать выигрыш в любой момент
- В случайный момент происходит краш
- Если игрок не забрал до краша - проигрывает ставку

### 3. UI/UX требования
- Современный минималистичный дизайн
- Центральный мультипликатор с анимацией роста
- Панель ставок с основными функциями
- История последних раундов
- Статистика игрока (баланс, прибыль)
- Адаптивный дизайн для разных платформ

### 4. Технические требования
- Unity 2022.3 LTS
- C# скрипты
- TextMeshPro для UI
- Unity UI система
- Аудио система
- Оптимизация для WebGL

### 5. Функциональность прототипа
- Базовая механика игры
- Правильная математика RTP 96%
- Основной UI интерфейс
- Система ставок и кешаута
- Визуальные эффекты и анимации
- Звуковые эффекты
- Система баланса игрока

### 6. Автофункции (базовые)
- Автоставки (фиксированная сумма)
- Автокешаут (целевой мультипликатор)
- 50% кешаут (частичный вывод)

### 7. Безопасность
- Provably Fair система
- Серверная валидация (симуляция)
- Защита от читов

## Критерии успеха
1. RTP точно 96% ± 0.3%
2. Плавная анимация роста мультипликатора
3. Отзывчивый UI на всех платформах
4. Стабильная работа на 60 FPS
5. Корректная математическая модель

## Ограничения
- Упрощенный прототип (не полная игра)
- Без мультиплеера (локальная игра)
- Без реальных денег (демо режим)
- Без сложных автофункций

## Технический стек
- Unity 2022.3 LTS
- C#
- TextMeshPro
- Unity UI
- Unity Audio System
- WebGL Build System
